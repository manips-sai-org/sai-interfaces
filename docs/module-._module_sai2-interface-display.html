Module: ./module/sai2-interface-display

Defines a custom HTML element to periodically query a Redis key at a certain frequency and displays it as a label. 
Element Tag: <sai2-interface-trajectory-display>
HTML attributes:
   key: string - Redis key to query
   refreshRate: number - How often, in seconds, to update
   decimalPlaces: number - How many decimal places to show
